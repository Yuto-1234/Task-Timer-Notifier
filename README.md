# Task Timer Notifier

タスク管理とタイマー機能を備えた、時間経過を音とポップアップで通知するアプリです。

## 実行方法

このアプリは React と Vite を使用しているため、ファイルを直接開くのではなく、開発サーバーを起動する必要があります。

1. 依存関係のインストール:
   ```bash
   npm install
   ```

2. 開発サーバーの起動:
   ```bash
   npm run dev
   ```

3. ブラウザで `http://localhost:3000` を開く。

## GitHub Pages へのデプロイ方法

1. ビルドの実行:
   ```bash
   npm run build
   ```

2. 生成された `dist` フォルダの中身を GitHub Pages にアップロードしてください。
